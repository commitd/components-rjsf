# Committed Components RJSF

<code>v{version}</code>

This is Committed's design system extension for React JSON Schema Forms.

## Getting Started

Install the library and peer-dependencies in your application

<code>npm add @committed/components-rjsf@{version}</code>

Pass the JSON Schema to the `JSForm` component and receive via the `onSubmit`.

```
import React from 'react'
import { JSForm } from '@committed/components-rjsf'

export const MyForm = () => {

  const handleSubmit =(e) => {
    // handle e.formData
  }

  return <JSForm schema={schema} onSubmit={handleSubmit}  />
}

```
